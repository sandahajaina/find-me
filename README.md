# Matcha – Full-Stack Dating App (Work in Progress)

## 📌 Description
Application web de type réseau social développée en full-stack.

## ⚙️ Stack
- Backend: Node.js (Express)
- Frontend: React
- Database: PostgreSQL
- DevOps: Docker, Docker Compose

## 🧱 Architecture
- Backend et frontend séparés
- Communication via API REST
- Base de données connectée
- Environnement conteneurisé

## 🚧 État actuel
- Docker & services configurés (backend, frontend, DB, adminer)
- Connexion base de données opérationnelle
- Structure backend en place
- Développement des routes en cours
- Authentification en cours d’implémentation

## 🎯 Objectifs
- Authentification sécurisée (JWT)
- Gestion des profils utilisateurs
- Matching & interactions
- Interface utilisateur complète

Renforcer mes compétences en développement full-stack (Node.js / React) et en architecture applicative.
